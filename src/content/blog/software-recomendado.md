## Stack Recomendado

### Editores de Código

| Editor | Lenguajes | Coste |
|--------|-----------|-------|
| VS Code | Todo | Gratis |
| JetBrains IDEs | Profesional | Pagado / OpenSource |
| Neovim | Expertos | Gratis |

### Terminal

- **Bash/Zsh** (macOS/Linux)
- **PowerShell 7** (Windows)
- **Prompt:** Oh My Zsh o Starship
- **Multiplexer:** Tmux o Zellij

### Versionado

```bash
git init
git remote add origin https://github.com/usuario/repo.git
git branch -M main
git push -u origin main
```

### Package Managers

- **Node:** npm, pnpm, yarn
- **Python:** pip, poetry
- **Rust:** cargo

### Bases de Datos

- **SQL:** PostgreSQL (recomendado)
- **NoSQL:** MongoDB, Redis
- **ORMs:** SQLAlchemy, Prisma
